  • Assignment 1 grades released

    Posted: 11/14/2024, 2:35:55 PM

    Hi all! We have released grades for the first practical assignment. There are a couple notes about the grading procedure I'd like to mention/reinforce:

    • The grades are pass/fail, and only the txt submission receives a grade. If you have received a grade for your .txt assignment but not for your .ipynb assignment, don't be alarmed - your assignment has been graded.
    • To receive a passing grade for the assignment, you have put reasonable effort into every task. Examples of reasonable effort include describing your solution in pseudocode even though you weren't able to debug your python code, or providing ideas about what a theoretical concept might mean even if you're not sure about them. 
    • The deadlines for the assignments are firm. If personal/medical circumstances prevent you from making the deadline, you must contact your student advisor, as we are not in position to grant deadline extensions.

    Overall the vast majority of students who submitted a notebook received a passing grade. Best of luck with the following assignments!

  • Assignment 4 possible error on Windows

    Posted: 11/13/2024, 1:57:07 PM

    Dear students 

    This is regarding assignment 4.

    If you get the error "ERROR: Invalid requirement /s" when installing the schnapsen environment on Windows then please update  the schnapsen install  command to be:

    !pip uninstall schnapsen -y && rd /s /q schnapsen && git clone https://github.com/intelligent-systems-course/schnapsen && cd schnapsen && pip install -e . && cd ..

    If you still get error try making an empty folder called schnapsen next to your notebook file. 

    Kind regards

    Pawel Piwowarski
